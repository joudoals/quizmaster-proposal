Class Product Proposal: QuizMaster

1. Product Name

QuizMaster - An online quiz application designed for students and educators to create, take, and manage quizzes efficiently.

2. Team Members

  • Joud - Scrum Master
  • Roudha - Product Owner

3. Application Overview

QuizMaster allows users to create quizzes with multiple-choice questions, take quizzes, and view their scores. Educators can manage quizzes and track student performance, while students can access a user-friendly interface to participate in quizzes. The application aims to enhance learning through interactive assessments.

4. Technology Overview

Languages: HTML, CSS, JavaScript, Node.js
Libraries/Frameworks: Bootstrap (for styling), Express (for backend), React (for future frontend enhancements)
Platforms: Azure Static Web Apps (hosting), Azure DevOps (project management)
Hosting: Hosted on Microsoft Azure

5. Proposal Last Updated

March 26, 2025

6. Feature List (Prioritized)

  1. Create a quiz with multiple-choice questions (5 story points)
  2. Take a quiz and submit answers (3 story points)
  3. View quiz results and scores (3 story points)
  4. User authentication (login/register) (5 story points)

7. Future Feature List

  • Real-time quiz analytics for educators
  • Gamified leaderboards for students
  • Integration with payment gateways for premium features
  • Mobile app support

8. Basic Technical Features

  • REST API for quiz creation and submission (learned in class)
  • JSON for data exchange between client and server
  • Unit testing with Jest for backend validation

9. Advanced Technical Features

  • Cross-Origin Resource Sharing (CORS) to handle API requests
  • Azure hosting with CI/CD pipeline using GitHub Actions
  • Scalable database design using Azure Cosmos DB
